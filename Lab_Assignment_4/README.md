# Community Engagement AI Assistant Frontend

This workspace contains only the Exercise 1 frontend for Lab 4.

## Stack

- React + Vite
- Apollo Client
- React Bootstrap

## Frontend-only notes

- No backend code is included.
- By default, the UI runs in mock mode so the chatbot can be demonstrated without a GraphQL service.
- To connect a real API later, copy `.env.example` to `.env`, set `VITE_GRAPHQL_URL`, and switch `VITE_USE_MOCK_CHAT=false`.

## Commands

```bash
npm install
npm run dev
```
